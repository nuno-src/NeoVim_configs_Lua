# NeoVim_configs_Lua
NeoVim configs in Lua

with:
- packer.nvim
- nvim-tree
- nvim-treesitter
- Lualine
- catppuccin

# How to Use:
Just copy `int.lua` and the `lua folder` to the neovim config folder `~/.config/nvim/`

# Install plugins:
Open `~/.config/nvim/lua/core/plgins.lua` with NeoVim and use `:PackerSync` command
